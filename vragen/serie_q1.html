informatie

Je ziet hier een batterij, drie schakelaars en een lamp. De schakelaars zijn in serie geschakeld. Bij een serie schakeling wordt gebruik gemaakt van een logische "en" operator. De lampen kunnen alleen aan gaan als en schakelaar 1 en schakelaar 2 en schakelaar 3 aan staan.

opdracht

Kun je de code zo aanpassen dat de lamp aan gaat als alle drie de schakelaars aan staan?

tips

  • De meeste code is reeds aanwezig, er mist nog een klein detail, zoek in het deel tussen de <script> tags om de aanpassing te maken.
  • Je zult een aanpassing moeten maken in de checkLamp() functie, wellicht heb je een if ... else constructie nodig?
  • In veel programmeertalen gebruik je een operator om een logische "en" uit te voeren, meestal ziet deze operator eruit als een & teken (enkel of dubbel uitgevoerd).
